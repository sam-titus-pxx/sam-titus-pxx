Tutorial #3-A:
How To Create A Multi-Layered Card-Box Container

1st Row 1st Column

2nd Row 1st Column



STEP 1: Creating A Row Container

1• Inside The Main Holding Container, Create Another Box Card: 


      <div 
      style="height:250px; width:100vw; 
      padding:0;" 
      class="w3-container w3-cell 
      w3-card w3-border">
      
      <div style="" class=""> </div>
      
      </div>

2• Set The Specificities of The Second Card Box:
class="w3-container w3-cell-row w3-blue" 


      <div 
      style="height:250px; width:100vw; 
      padding:0;" 
      class="w3-container w3-cell 
      w3-card w3-border">
      
      <div style="" class="w3-container 
      w3-cell-row w3-blue"> </div>
      
      </div>



3• Set The Styling of The Second Card Box:
style="width:100%;" 


      <div 
      style="height:250px; width:100vw; 
      padding:0;" 
      class="w3-container w3-cell 
      w3-card w3-border">
      
      <div style="width:100%;"
      class="w3-container 
      w3-cell-row w3-blue"> </div>
      
      </div>



4• Create Another Child Card Box Inside The Main Holding Container:
Set Its Color To Black 


      <div 
      style="height:250px; width:100vw; 
      padding:0;" 
      class="w3-container w3-cell 
      w3-card w3-border">
      
      <div style="width:100%;"
      class="w3-container 
      w3-cell-row w3-blue"> </div>
      
      <div style="width:100%;"
      class="w3-container 
      w3-cell-row w3-black"> </div>
      
      </div>



5• Add Text To The Card Boxes: 


      <div 
      style="height:250px; width:100vw; 
      padding:0;" 
      class="w3-container w3-cell 
      w3-card w3-border">
      
      <div style="width:100%;"
      class="w3-container 
      w3-cell-row w3-blue"> 
         <h1>1st Row Box</h1>
      </div>
      
      <div style="width:100%;"
      class="w3-container 
      w3-cell-row w3-black">
         <h1>2nd Row Box</h1>
      </div>
      
      </div>
