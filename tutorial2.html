How To Create Web Links Using Icons

facebook twitter instagram


1• Create A Span Tag: 


      <span class="">
      </span>

2• Set Span Style Specificity:
• Setting color and inner padding 


      <span class="w3-blue pd-sm">
      </span>

3• Include The FA5 Icon Of Choice:
• For facebook icon: fab fa-facebook 


     <span class="w3-blue pd-sm
      fab fa-facebook">
      </span>
• For twitter icon: fab fa-twitter
• For instagram icon: fab fa-instagram
• For whatsapp icon: fab fa-whatsapp
• For telegram icon: fab fa-telegram

4• Create The Link Tag Inside Span: 


      <span class="w3-blue pd-sm
      fab fa-facebook">
      
      <a href="">
      </a>
      
      </span>

5• Specify The Target URL: 


     <span class="w3-blue pd-sm
      fab fa-facebook">
      
      <a href="www.facebook.com">
      </a>
      
      </span>

6• Put A Text Inside The Link Tag: 


      <span class="w3-blue pd-sm
      fab fa-facebook">
      
      <a href="www.facebook.com">
      facebook
      </a>
      
      </span>
